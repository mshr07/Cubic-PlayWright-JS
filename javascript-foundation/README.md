# JavaScript Foundation

This folder is a mini learning project for JavaScript concepts that matter directly in Playwright automation.

## Why This Comes First

Playwright test authors constantly use:

- variables and data types
- functions and callbacks
- loops and conditions
- objects and arrays
- async and `await`
- JSON and reusable helpers
- DOM concepts used by browser automation

If these are weak, Playwright feels hard for the wrong reason.

## How To Use This Folder

- Read the comments in each file
- Run selected files with `node <file>`
- Open DOM pages from `dom-pages/` in a browser if you want to inspect the HTML manually
- Compare each topic with how it later appears in Playwright tests

## Learning Outcomes

By the end of this folder, a learner should be able to:

- write clean JavaScript for UI and API automation
- understand async Playwright commands
- use objects, arrays, and JSON for test data
- create reusable helpers
- avoid common beginner mistakes
