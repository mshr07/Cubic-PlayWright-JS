# JavaScript Foundation Roadmap

## Stage 1: Core Syntax

Start with:

- `01-basics/variables.js`
- `01-basics/primitive-data-types.js`
- `01-basics/reference-types.js`
- `01-basics/template-literals.js`

Goal: get comfortable storing and printing values.

## Stage 2: Writing Logic

Continue with:

- `02-functions/`
- `03-control-flow/`
- `04-arrays/`
- `05-objects/`

Goal: write readable logic that looks like real test code.

## Stage 3: How JavaScript Behaves

Study:

- `06-scope-and-hoisting/`
- `07-modules/`
- `08-error-handling/`
- `09-asynchronous-javascript/`

Goal: understand why JavaScript behaves the way it does in automation code.

## Stage 4: Browser and Data Basics

Study:

- `10-dom-basics/`
- `11-json-and-data-handling/`

Goal: connect JavaScript with browser pages and API-style responses.

## Stage 5: Tester-Focused Practice

Finish with:

- `12-practical-js-for-testers/`

Goal: write reusable automation-friendly JavaScript with clean helper patterns.
